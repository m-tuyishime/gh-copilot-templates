# Examples

This folder contains real-world examples showing the template files in action with specific customizations.

## What's Here

These are the **personalized** versions of the templates, demonstrating how to adapt the generalized templates from `/templates` to a specific project with:
- Actual tech stack details
- Specific conventions and rules
- Real workflow patterns
- Concrete examples

## Examples Included

### `/custom-prompts`
Personalized prompt files showing specific workflows:
- Git commit conventions with atomic commit practices
- Convention capture automation with specific instruction files listed

### `/instruction-files`
Complete instruction file setup for a Next.js/React/TypeScript project:
- `AGENTS.md` with React Server Components, TypeScript strict mode, Tailwind CSS
- `copilot-instructions.md` with VS Code-specific workflows
- `tests.instructions.md` for testing conventions
- `docs.instructions.md` for documentation standards
- `dal.instructions.md` for data access layer patterns

## How to Use These Examples

1. **Study** the examples to see how the templates are customized
2. **Compare** with the templates to understand what was changed
3. **Adapt** the patterns to your own project's needs
4. **Copy** specific sections that match your use case

## Need Templates?

Check the `/templates` folder for generalized versions you can customize from scratch.
